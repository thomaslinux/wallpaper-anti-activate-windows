# Wallpaper to hide Activate Windows

A wallpaper the same color as the text saying Activate Windows,

so you can't see the text Activate Windows when on the desktop.

![](activate-windows-miniature.png)

# How to use / Instructions

**Once :**

Right click, set as wallpaper.

**Always :**

Right click, download on desktop, set as wallpaper from the desktop.
